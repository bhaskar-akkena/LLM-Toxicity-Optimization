Detailed Description

Abstract base class for all variation operators (e.g., mutation, crossover). 

Abstract base class for all variation operators (mutation, crossover, etc.).
Each operator must implement the `apply()` method, which returns a modified string.

Each variation operator must implement the apply() method, which defines how the input string is modified. Additional metadata can be attached for experiment tracking.
