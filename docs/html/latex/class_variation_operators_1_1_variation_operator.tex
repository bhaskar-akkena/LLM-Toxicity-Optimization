\doxysubsection{Detailed Description}
Abstract base class for all variation operators (e.\+g., mutation, crossover). 

\begin{DoxyVerb}Abstract base class for all variation operators (mutation, crossover, etc.).
Each operator must implement the `apply()` method, which returns a modified string.
\end{DoxyVerb}


Each variation operator must implement the {\ttfamily \doxylink{class_variation_operators_1_1_variation_operator_a37e9335417542454bea5b151a0f373a6}{apply()}} method, which defines how the input string is modified. Additional metadata can be attached for experiment tracking.
