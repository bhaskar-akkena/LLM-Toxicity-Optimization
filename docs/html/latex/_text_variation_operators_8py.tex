\doxysubsection{Detailed Description}
Contains concrete mutation operators for prompt-\/level variations. 

This module defines various prompt mutation operators like synonym replacement, deletion, paraphrasing, masked language modeling, and back-\/translation. All operators implement the Variation\+Operator interface.
